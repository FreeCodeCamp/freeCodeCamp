# --description--

When the value associated with a key is an object, the `get` method will return a reference to that object. You can use that reference to modify the object inside the map.

To test it, use the `getBallerina` reference and set the `small` pots to another integer. Then, look at the developer console to check that the value has changed in your catalog.
