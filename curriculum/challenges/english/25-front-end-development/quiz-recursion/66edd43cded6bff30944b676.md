### --question--

#### --text--

Which of the following is an example of recursion?



#### --distractors--

```javascript
function sumArray(arr) {
    return arr[0] + sumArray(arr.slice(1));    
}
```

---

```javascript
function countDown(n) {
    if (n === 0) return;
    n--;
    countDown(n);
}
```

---

```javascript
function findMax(arr) {
   if (arr.length === 1) return arr[0];
   return findMax(arr);
}
```

#### --answer--

```javascript
function countdown(n) {
    if (n === 0) return 0;
    return n + countdown(n - 1);
}
```
