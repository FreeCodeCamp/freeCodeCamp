## --text--

What is wrong with the following XML?:

```xml
<person>
  <name>Chuck</name>
  <phone type="intl">
    +1 734 303 4456
  <email hide="yes" />
</person>
```

## --distractors--

Email tag is missing closing tag.

---

Spacing will cause XML to be invalid.

---

Plain text should be encoded using UTF-8.

## --answer--

Phone tag is missing closing tag.
